Architecture Components

Client Layer: React frontend with Vite
Reverse Proxy: Nginx routing
API Layer: Express.js backend
AI Services: Azure OCR & Groq LLM
Data Layer: MongoDB database
Infrastructure: Ubuntu VM with PM2

Key Specifications

  • Frontend: React 18 with Vite, running on port 4173
  • Backend: Express.js with Node.js 20 LTS, running on port 5000
  • Database: MongoDB 7.0+ with optimized indexes
  • OCR: Azure Document Intelligence with 85%+ accuracy
  • AI/LLM: Groq API for intelligent medication parsing
  • File Upload: Multer with 10MB limit and MIME validation
  • Authentication: JWT tokens with bcryptjs hashing, role-based access control
  • Infrastructure: Ubuntu 24.04 LTS VM (2 vCPU, 4GB RAM) with PM2 process management
  • Reverse Proxy: Nginx for SSL/TLS and request routing
